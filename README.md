# This app is developed in django 4

# To Get Started 
 - git clone https://github.com/ibrahmad18/alhouda_online.git
0. Requirements
   - Python 3.8 minimum
   - Django 4.2 minimum

1. Clone the Repo 

2. Migrate Db
  - cd project
  - python manage.py makemigrations
  - python manage.py migrate

3. Run app
   - python manage.py runserver
